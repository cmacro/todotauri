Todo -Leptos - Tauri
====================

## Introduction
Tauri2 Desktop app

## Preparation

- cargo install trunk
- cargo install tauri-cli
- rustup target add wasm32-unknonwn-unknown

## Create

`cargo new todo-tauri`

## Todo Leptos
adding the dependency to cargo.toml
```toml
[dependencies]
leptos = { version = "0.7.5", features = ["csr"]}

```
